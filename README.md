# Application de gestion des salles

## **Auteur :** Idir Islam Chili

### Description :
Cette application permet de gérer des salles (ajout, modification, suppression, recherche et affichage).
Elle est développée en Python avec une interface graphique (CustomTkinter) et une base de données MySQL.

### Architecture:
Le projet est structuré en couches :

Data : accès aux données (DAO)
Models : entités métier
Services : logique métier et validation
Views : interface graphique
main.py : point d’entrée

### ⚙️ Technologies utilisées :
Python, 
MySQL, 
CustomTkinter.
